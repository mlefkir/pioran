psd_base#

Base representation of a power spectral density function. It is not meant to be used directly, but rather as a base class to build PSDs. The sum and product of PSD are implemented with the + and * operators, respectively.

Overview#

Classes#

Represents the product of two power spectral densities.

Represents the sum of two power spectral densities.

Represents a power density function function.

Bridge between the parameters and the power spectral density function. All power spectral density functions inherit from this class.

Values of the parameters of the power spectral density function.
