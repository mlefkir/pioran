psdtoacv#

Convert a power spectral density to an autocovariance function via the inverse Fourier transform methods or kernel decomposition.

class pioran.psdtoacv.PSDToACV(PSD: pioran.psd_base.PowerSpectralDensity, S_low: float, S_high: float, T: float, dt: float, method: str, n_components: int = 0, estimate_variance: bool = True, init_variance: float = 1.0, use_celerite=False, use_legacy_celerite: bool = False)[source]#

Bases: equinox.Module

Represents the tranformation of a power spectral density to an autocovariance function.

Computes the autocovariance function from a power spectral density using the several methods.

Parameters:
PSDPowerSpectralDensity

Power spectral density object.

S_lowfloat

Lower bound of the frequency grid.

S_highfloat

Upper bound of the frequency grid.

Tfloat

Duration of the time series.

dtfloat

Minimum sampling duration of the time series.

methodstr

Method used to compute the autocovariance function. Can be ‘FFT’ if the inverse Fourier transform is used or ‘NuFFT’ for the non uniform Fourier transform. The ‘SHO’ method will approximate the power spectral density into a sum of SHO functions.

n_componentsint

Number of components used to approximate the power spectral density using the ‘SHO’ method.

estimate_variancebool, optional

If True, the amplitude of the autocovariance function is estimated. Default is True.

init_variancefloat, optional

Initial value of the variance. Default is 1.0.

Raises:
TypeError

If PSD is not a PowerSpectralDensity object.

ValueError

If S_low is smaller than 2., if method is not in the allowed methods or if n_components is smaller than 1.

decompose_model(psd_normalised: jax.Array)[source]#

Decompose the PSD model into a sum of basis functions.

Assuming that the PSD model can be written as a sum of \(J\) , this method solve the linear system to find the amplitude \(a_j\) of each kernel.

(1)#\[\]

boldsymbol{y} = B boldsymbol{a}

with \(\boldsymbol{y}=\begin{bmatrix}1 & \mathcal{P}(f_1)/\mathcal{P}(f_0) & \cdots & \mathcal{P}(f_J)/\mathcal{P}(f_0) \end{bmatrix}^\mathrm{T}\) the normalised power spectral density vector, \(B\) the spectral matrix associated to the linear system and \(\boldsymbol{a}\) the amplitudes of the functions.

(2)#\[\]

B_{ij} = dfrac{1}{1 + left(dfrac{f_i}{f_j}right)^4}

(3)#\[\]

B_{ij} = dfrac{1}{1 + left(dfrac{f_i}{f_j}right)^6}

Parameters:
psd_normalisedjax.Array

Normalised power spectral density by the first value.

Returns:
jax.Array

Amplitudes of the functions.

jax.Array

Frequencies of the function.

calculate(t: jax.Array, with_ACVF_factor: bool = False) → jax.Array[source]#

Calculate the autocovariance function from the power spectral density.

The autocovariance function is computed by the inverse Fourier transform by calling the method get_acvf_byFFT. The autocovariance function is then interpolated using the method interpolation.

get_cov_matrix(xq: jax.Array, xp: jax.Array) → jax.Array[source]#

Compute the covariance matrix between two arrays xq, xp.

The term (xq-xp) is computed using the EuclideanDistance() function from the utils module. If the method used is ‘NuFFT’ and if the two arrays have the same shape, the covariance matrix is computed only on the unique values of the distance matrix using the decompose_triangular_matrix() and reconstruct_triangular_matrix() functions from the utils module. Otherwise, the covariance matrix is computed on the full distance matrix.

Parameters:
xqjax.Array

First array.

xpjax.Array

Second array.

Returns:
jax.Array

Covariance matrix.

NotImplementedError

If the method is not implemented.
