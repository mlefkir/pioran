acvf_base#

Base representation of a covariance function. It is not meant to be used directly, but rather as a base class to build covariance functions. The sum and product of covariance functions are implemented with the + and * operators, respectively.

Classes#

Represents a covariance function model.

Bridge between the parameters and the covariance function model. All covariance functions inherit from this class.
