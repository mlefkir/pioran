class pioran.psd.Lorentzian(parameters_values: list, free_parameters: list = [True, True, True])[source]#

Bases: pioran.psd_base.PowerSpectralDensity

Lorentzian power spectral density.

(1)#\[\mathcal{P}(f) = \dfrac{A}{\gamma^2 +4\pi^2 (f-f_0)^2}.\]

with the amplitude \(A\ge 0\), the position \(f_0\ge 0\) and the halfwidth \(\gamma>0\).

The parameters are stored in the parameters attribute which is a ParametersModel object. The values of the parameters can be accessed using the parameters attribute via three keys: ‘position’, ‘amplitude’ and ‘halfwidth’.

The power spectral density function is evaluated on an array of frequencies \(f\) using the calculate method.

Parameters:
param_valueslist of float

Values of the parameters of the power spectral density function. [position, amplitude, halfwidth]

free_parameters: :obj:`list` of :obj:`bool`, optional

List of bool to indicate if the parameters are free or not. Default is [True, True,True].

class pioran.psd.Gaussian(parameters_values, free_parameters=[True, True, True])[source]#

Bases: pioran.psd_base.PowerSpectralDensity

Gaussian power spectral density.

(2)#\[\mathcal{P}(f) = \dfrac{A}{\sqrt{2\pi}\sigma} \exp\left(-\dfrac{\left(f-f_0\right)^2}{2\sigma^2} \right).\]

with the amplitude \(A\ge 0\), the position \(f_0\ge 0\) and the standard-deviation ‘sigma’ \(\sigma>0\).

The parameters are stored in the parameters attribute which is a ParametersModel object. The values of the parameters can be accessed using the parameters attribute via three keys: ‘position’, ‘amplitude’ and ‘sigma’

The power spectral density function is evaluated on an array of frequencies \(f\) using the calculate method.
