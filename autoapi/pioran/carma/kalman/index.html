Predict(X, P, F, Q)[source]#

Predict step of the Kalman filter.

Given the state vector \(\boldsymbol{X}_k\) and the covariance matrix \(\boldsymbol{P}_k\) at time \(t_k\), this method computes the predicted state vector \(\hat{\boldsymbol{X}}_{k+1}\) and the predicted covariance matrix \(\hat{\boldsymbol{P}}_{k+1}\) at time \(t_{k+1}\). Using the notation of the Kalman filter, this method computes the following equations:

(1)#\[\begin{split}\hat{\boldsymbol{X}}_{k+1} &= {F}_k \boldsymbol{X}_k \\ \hat{\boldsymbol{P}}_{k+1} &= {F}_k {P}_k {F}_k^\mathrm{T} + {Q}_k\end{split}\]

where \({F}_k\) is the transition matrix and \({Q}_k\) is the covariance matrix of the noise process.

Parameters:
Xjax.Array

State vector.

Pjax.Array

Covariance matrix of the state vector.

Fjax.Array

Transition matrix.

Qjax.Array

Covariance matrix of the noise process.

Returns:
Xjax.Array

Predicted state vector.

Pjax.Array

Covariance matrix of the predicted state vector.

Update(X, P, Z, H, R)[source]#

Update step of the Kalman filter.

Given the predicted state vector \(\hat{\boldsymbol{X}}_{k+1}\) and the predicted covariance matrix \(\hat{{P}}_{k+1}\) at time \(t_{k+1}\), this method computes the updated state vector \(\boldsymbol{X}_{k+1}\), the updated covariance matrix \({P}_{k+1}\), the measurement residual \(\boldsymbol{Y}_{k+1}\) and the innovation covariance matrix \({S}_{k+1}\) at time \(t_{k+1}\).

Using the notation of the Kalman filter, this method computes the following equations:

(2)#\[\begin{split}\boldsymbol{Y}_{k+1} &= \boldsymbol{Z}_{k+1} - {H}_{k+1} \hat{\boldsymbol{X}}_{k+1} \\ {S}_{k+1} &= {H}_{k+1} \hat{{P}}_{k+1} {H}_{k+1}^\mathrm{T} + {R}_{k+1} \\ {K}_{k+1} &= \hat{{P}}_{k+1} {H}_{k+1}^\mathrm{T} {S}_{k+1}^{-1} \\ \boldsymbol{X}_{k+1} &= \hat{\boldsymbol{X}}_{k+1} + {K}_{k+1} \boldsymbol{Y}_{k+1} \\ {P}_{k+1} &= ({I} - {K}_{k+1} {H}_{k+1}) \hat{{P}}_{k+1}\end{split}\]
Parameters:
Xjax.Array

Predicted state vector.

Pjax.Array

Covariance matrix of the predicted state vector.

Zjax.Array

Observation vector.

Hjax.Array

Observation matrix.

Rjax.Array

Covariance matrix of the observation noise.

one_step_loglike_CAR1(carry, xs)[source]#

Compute the log-likelihood of a single observation value.

This function is used in the log_likelihood() method to compute the sequentially the log-likelihood of all the observations values. It is called using the jax.lax.scan() function. This function calls the Predict() and Update() methods. The one-step log-likelihood is computed using the following equation:

(3)#\[\log p(\boldsymbol{Z}_k|\boldsymbol{Z}_{1:k-1}) = -\frac{1}{2} \log |{S}_k| - \frac{1}{2} \boldsymbol{Y}_k^\mathrm{T} {S}_k^{-1} \boldsymbol{Y}_k\]
Parameters:

Tuple containing the state vector \(\boldsymbol{X}_k\), the covariance matrix \(\boldsymbol{P}_k\) and the log-likelihood \(\log p(\boldsymbol{Z}_k|\boldsymbol{Z}_{1:k-1})\) at time \(t_k\).

xstuple

Tuple containing the time increment \(\Delta t_k\), the observation value \(\boldsymbol{Z}_k\) and the observation error \(\boldsymbol{\epsilon}_k\) at time \(t_k\).

Returns:
carrytuple

Tuple containing the state vector \(\boldsymbol{X}_{k+1}\), the covariance matrix \(\boldsymbol{P}_{k+1}\) and the log-likelihood \(\log p(\boldsymbol{Z}_{k+1}|\boldsymbol{Z}_{1:k})\) at time \(t_{k+1}\).

xstuple

Tuple containing the time increment \(\Delta t_{k+1}\), the observation value \(\boldsymbol{Z}_{k+1}\) and the observation error \(\boldsymbol{\epsilon}_{k+1}\) at time \(t_{k+1}\).
